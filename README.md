# *cattive-portal*
An evil captive portal designed to work on a Raspberry Pi set up as an indipendent wireless AP.

## How to
Install Flask with:

    pip install flask

Set up the portal by editing the configuration file:

    nano portal.conf

Start the web-app:

    python portal.py
